# Pi Agent Web UI 成本分析

## 💰 必然花费：AI 模型 API 调用

无论有没有 Web UI，只要发消息给 AI，API 费用就跑不掉。

| 档次 | 模型示例 | 大致价格 |
|------|---------|---------|
| 便宜 | DeepSeek、OpenCode Zen | 极低 / 有免费额度 |
| 中等 | Claude Sonnet、GPT-4o | 几美元 / 百万 token |
| 较贵 | Claude Opus | 十几美元 / 百万 token |

> 一个典型开发 session 消耗几万～十几万 token，费用约几美分到几美元。

---

## 🆓 免费部分

| 项目 | 说明 |
|------|------|
| Pi Agent SDK | MIT 开源，完全免费 |
| pi-web（前端） | MIT 开源，开箱即用 |
| 本地运行 | `localhost` 上跑，零成本 |
| Claude Code Router | 本地路由服务，免费 |
| 开发调试 | Next.js dev server，本地运行不花钱 |

---

## 💸 公网部署（可选）

只有部署到公网给别人用时才会产生：

| 项目 | 大致费用 |
|------|---------|
| 服务器托管 | Vercel / Netlify 有免费额度，小项目够用 |
| 域名 | 几十元 / 年 |
| 用户认证 | Clerk / Auth0 有免费额度 |

---

## 📊 总结

| 场景 | 成本 |
|------|------|
| **本地自己用** | ≈ API 调用费（几美分～几美元/次） |
| **部署公网分享** | API 费 + 可能一点服务器费用 |
| **SDK / UI 代码** | 完全免费（MIT） |
