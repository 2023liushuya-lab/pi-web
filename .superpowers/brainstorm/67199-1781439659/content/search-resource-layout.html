搜索 + 资源库 — 布局方案

搜索用 Cmd+P 弹窗，资源库放侧边栏

方案 A：侧边栏新增"资源库"标签

在现有 Sessions / Files 旁边加一个 Resources 标签

侧边栏结构
🔍
Sessions Files 📦 Resources
📁 Session 1
📁 Session 2
📁 Session 3

优点

  • 复用现有标签栏模式，改动最小
  • 用户认知负担低

缺点

  • 三个标签略显拥挤
  • 资源库与 Sessions/Files 平级可能层级不清

方案 B：侧边栏底部独立入口

资源库作为侧边栏底部的固定图标入口

侧边栏结构
Sessions Files
📁 Session 1
📁 Session 2
📁 Session 3
📦 资源库 12 个文件

优点

  • Sessions / Files 保持简洁
  • 资源库有独立视觉层级

缺点

  • 需要打开子面板才能浏览文件
  • 入口不够显眼，可能被忽略
