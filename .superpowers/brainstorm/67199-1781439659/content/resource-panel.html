资源库交互细节

点击文件后

  • 在右侧打开 FileViewer 标签页（复用现有文件查看器）
  • 显示文件内容 + "来自 [Session 名称]" 面包屑
  • 可点击面包屑跳转到对应对话

分组排序方式

  • 底部切换：按类型 / 按时间 / 按来源会话
  • 默认按时间倒序（最新在前）

技术方案

API GET /api/resources?sort=time|type|session
数据来源 解析 .jsonl 中的 toolCall 写文件记录 + 扫描会话 cwd 下新文件
文件查看 复用 /api/files/[...path]
组件 新增 ResourcePanel.tsx，嵌入 SessionSidebar
